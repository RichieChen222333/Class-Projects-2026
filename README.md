# 🚀 2026 全端與 AI 開發班 - 學生作品集

這是一個匯集了本屆學員心血的專案牆。從前端介面設計、AI 工具應用到後端系統整合，每一份作品都代表了學員在職業訓練過程中的成長與蛻變。

---

## 🛠️ 技術棧 (Tech Stack)
本班級課程涵蓋以下技術，並應用於各項作品中：
* **AI 應用：** ChatGPT, Midjourney, Claude, AI Code Assistant
* **前端開發：** HTML5, CSS3, JavaScript (ES6+), Bootstrap 5, Vue.js
* **後端開發：** PHP (Laravel) / Python (Flask)
* **資料庫：** MySQL / PostgreSQL
* **部署與工具：** Git/GitHub, Render, Docker

## 📢 繳交說明
如果你是本班學員，請遵循以下步驟提交作品：
1. **Fork** 本專案。
2. 在 `students/` 下建立以自己「學號-姓名」命名的資料夾。
3. 將你的程式碼放入該資料夾。
4. 修改此 `README.md` 的表格，新增你的一列資訊。
5. 發起 **Pull Request** 等待導師審核。
